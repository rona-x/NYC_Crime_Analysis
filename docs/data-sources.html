New York Crime Analysis

Chapter 2 Data sources

The following two data sets are both directly exported as CSV files from NYC Open Data.

2.1 Crime data

It is a large data set which contains 35 columns and 7.38M rows. The problem with this data set lies in processing it ready for our analysis, since it might involve irrelevant information, missing values and unclear structure.

Link: https://data.cityofnewyork.us/Public-Safety/NYPD-Complaint-Data-Historic/qgea-i56i

Collected by: Zhisen Cai

2.2 Hate crime data

This data set has 14 columns and 1,181 rows, which is a more manageable size compared to the previous one. The only problem with it is missing values.

Link: https://data.cityofnewyork.us/Public-Safety/NYPD-Hate-Crimes/bqiq-cu78

Collected by: Rona Xu
